## ⚡️ Getting Started

### 1. Clone & Install

```bash
git clone <repository-url>
cd "tugas react 24090055"
npm install
```

### 2. Development

```bash
npm run dev
# Open http://localhost:5173
```

### 3. Production Build

```bash
npm run build
npm run preview
# Open http://localhost:4173
```

### 4. Deployment (Vercel)

1. Push ke repository GitHub
2. Hubungkan ke [Vercel](https://vercel.com/import)
3. Set output directory: `dist`
4. Deploy!
